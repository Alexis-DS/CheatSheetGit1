Quelques commandes de bases.

    • 3.Quelques sites pour plus de commandes.

    git clone : Pour cloner un dépôt existant, c’est-à-dire, le récupérer sur votre machine

    git status : Pour connaître les fichiers modifiés récemment

    git diff : Pour voir ce qui a été changé

    git add fichier1 fichier2 : Pour ajouter des fichiers à prendre en compte lors du commit

    git commit : Pour faire un commit, ie sauvegarder vos changements

    git commit -a : Inclure tous les fichiers listés dans git status

    git commit fichier1 fichier2 : Indiquer précisément quels fichiers inclure

    git log : Visionner les logs

    git commit –amend : Options utiles: -p ou –statModifier le dernier message de commit

    git checkout fichier : Restaurer un fichier à son état du dernier commit

    git pull : Récupérer les nouveautés présentes sur le serveur

    git push : Envoyer ses propres modifications au serveur

    git revert idDuCommit : Annulé un commit publié


    Les branches


    git branch : Voir les branches (l'étoile montre sur laquelle on se trouve)

    git branch nomBranche : Créer une branche

    git checkout nomBranche : Se placer dans une branche

    git merge nomBranche : Fusionner une branche dans la branche où vous vous trouvez

    git branch -d nomBranche : Supprimer une branche
